Api para buscar los repetidores con la id de las unidades


    Login para poder obtener el sid, una vez que obtenga el sid ya puedo hacer cualquier peticion a la Api de Wialon

    https://hst-api.wialon.com/wialon/ajax.html?svc=token/login
    params{"token":"ya tengo el token"}

    Repetidor
    Metodo HTTPS: POST
    https://hst-api.wialon.com/wialon/ajax.html?svc=core/search_items&params={"spec":{"itemsType":"avl_retranslator","propName":"sys_name","propValueMask":"*","sortType":"sys_name"},"force":1,"flags":16777215,"from":0,"to":0}&sid="generado con el token de wialon"
